## DutyBoard
Small application to check current duty of developers in team

### What it really is?
This is light web app to control passing duty of presentations and test-checking duties between developers in team.

### Technologies
* JDK 15
* SpringBoot 2
* Thymeleaf + HTML
* Maven
##### DataSources:
* MySQL
* SQLite
* MongoDB
* JSON file
* Text file

### Packaging
Single JAR file
